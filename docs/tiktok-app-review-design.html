1. Application Overview

Kelasify is a multi-channel management platform serving Malaysian e-commerce sellers. The TikTok Shop integration enables sellers to manage their TikTok Shop operations from a centralized dashboard — syncing analytics, financial data, and affiliate performance into actionable insights.

4. Key Use Cases
  • 1
    Performance Monitoring — Sellers view their TikTok Shop's overall performance (total orders, GMV, buyers, video views, conversion rates) from the Admin Dashboard's Analytics tab, enabling data-driven business decisions.
  • 2
    Content Performance Tracking — Track which videos drive the most engagement and sales. Per-video metrics (views, likes, comments, shares) are synced and linked to content items in the CMS for ROI analysis.
  • 3
    Financial Reconciliation — Access settlement statements and transaction details to reconcile TikTok Shop payouts, track revenue, platform fees, and adjustments in one centralized view.
  • 4
    Affiliate Program Management — View a leaderboard of affiliate creators ranked by GMV, monitor creator-generated orders and commissions, and evaluate creator content performance for collaboration decisions.
  • 5
    Product Performance Analysis — Compare product-level metrics (impressions, clicks, orders, conversion rates) to identify top-performing products and optimize catalog strategy on TikTok Shop.

7. Security & Data Privacy
  • OAuth 2.0 tokens encrypted at rest (AES-256)
  • Automatic token refresh before expiry
  • HTTPS enforced for all API communication
  • Role-based access control (Admin only)
  • API credentials stored in secure vault table
  • All API responses logged for audit trail
  • CSRF protection on all forms
  • Data access limited to authorized shop only
